# CodeAlpha E-Commerce Store

A full-stack E-Commerce Store built using:

## Tech Stack

- HTML
- CSS
- JavaScript
- Node.js
- Express.js
- MongoDB

## Features

- Product Listings
- Product Details Page
- Shopping Cart
- User Registration/Login
- JWT Authentication
- Order Processing
- MongoDB Database Storage

## Run Locally

```bash
npm install
npm start
```

Server runs at:

```text
http://localhost:5000
```
